# C4 Architecture

- Actors: End Users, Developers, Admins.
- System: Modular Component Showcase Application.

## Container Level
- React Frontend
- FastAPI Gateway
- Node/Express Backend
- Spring Boot Service
- PostgreSQL
- MongoDB

## Component Level (Backend)
- Routes
- Middleware
- Services
- SQL integration layer
- Mongo model layer
- Vector/hybrid search services

## Component Level (Gateway)
- Controllers
- Service clients
- Security dependencies
- Rate limiting/tracing/error middleware
