# Multi-Framework Comparison

## Node.js + Express
- Rapid API development and broad npm ecosystem.
- Used for domain-heavy component and engagement APIs.

## FastAPI
- Async ingress proxy and service composition.
- Used for API gateway and routing abstraction.

## Spring Boot
- Enterprise-grade security, JPA, actuator, and metrics.
- Used for strongly structured Java service layer.

## ASP.NET Core (awareness)
- Comparable enterprise backend framework with DI/middleware pipeline.
- Strong option in Microsoft-centric teams.

## Ownership Boundaries
- Node: primary API domain operations and hybrid persistence.
- Gateway: ingress, proxying, resilience controls.
- Spring: enterprise Java CRUD/security path.
