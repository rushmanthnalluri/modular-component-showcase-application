# Deployment Screenshot Placeholders

| Placeholder | Expected filename | Capture requirement |
|---|---|---|
| Prometheus | `prometheus-targets.png` | Prometheus targets page showing backend, gateway, and Spring targets healthy. |
| Grafana | `grafana-dashboard.png` | Dashboard with HTTP latency/error panels populated. |
| Jaeger | `jaeger-trace.png` | End-to-end request trace with service spans. |
| CI/CD | `cicd-pipeline.png` | Passing GitHub Actions workflow with quality, Spring, and release-readiness jobs. |
| Docker | `docker-compose-services.png` | Running local stack or Docker Desktop services view. |
| Kubernetes | `kubernetes-pods.png` | Pod or deployment state from `kubectl get pods` or dashboard. |
| Render | `render-dashboard.png` | Render service dashboard or deployment activity page. |
