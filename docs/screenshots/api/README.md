# API Screenshot Placeholders

| Placeholder | Expected filename | Capture requirement |
|---|---|---|
| Response schema | `api-response-schema.png` | Swagger/OpenAPI schema panel for a representative response payload. |
| Request tracing | `api-request-tracing.png` | Jaeger or correlated logs showing request flow across gateway and downstream services. |
| JWT flow diagram | `api-jwt-flow-diagram.png` | Diagram or whiteboard capture of token issue, forward, and authorize flow. |
| RBAC proof | `api-rbac-proof.png` | Endpoint access matrix or live request denying/allowing users by role. |
| Retry and circuit breaker | `api-retry-circuit-breaker.png` | Gateway resilience evidence showing timeout, retry, or open-circuit behavior. |
