# Microservices Screenshot Placeholders

| Placeholder | Expected filename | Capture requirement |
|---|---|---|
| Dependency graph | `microservice-dependency-graph.png` | Diagram of frontend, gateway, backend, Spring, SQL, Mongo, and observability edges. |
| Bounded context map | `bounded-context-map.png` | Context map showing ownership boundaries per service. |
| Saga flow diagram | `saga-flow-diagram.png` | Distributed workflow diagram with outbox/compensation steps. |
| Distributed trace | `distributed-trace-waterfall.png` | Jaeger waterfall or equivalent multi-hop trace. |
| Token forwarding | `token-forwarding-proof.png` | Header propagation or auth forwarding capture at gateway/service boundary. |
